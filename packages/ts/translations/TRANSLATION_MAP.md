## Translation Scopes

### `common`
Shared UI elements, buttons, and actions used across multiple apps.

### `navigation`
Navigation menu items.

### `errors`
Error messages.

### `validation`
Form validation messages.

### `dashboard`
Specific to the Next.js `web-dashboard` app.

### `sdks`
Specific to the SvelteKit `web-sdks-apps` app.

# Postman API Tests

This directory contains Postman collections for comprehensive API testing.

## Quick Start

1. Import `collection.json` into Postman
2. Start both backend servers:
   ```bash
   moon run api-nest:dev    # Port 3000
   moon run api-golang:dev  # Port 8080
   ```
3. Run the collection in Postman or use Newman CLI

See [README.md](./README.md) for detailed documentation.
